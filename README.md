# 🚀 Auto Tax Token (Ethereum)

An ERC20 token with automatic taxation built using Solidity and OpenZeppelin.

## 🔥 Tokenomics

- 5% total tax on every transfer
  - 2% Burn 🔥 (permanently removed from supply)
  - 3% Treasury 💰
- 95% goes to recipient
- Owner-controlled tax adjustments (max 10%)
- Built for Ethereum (Sepolia + Mainnet ready)
